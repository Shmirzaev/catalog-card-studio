# Catalog Card Studio — Milana Premium

A browser-based generator for Milana Premium product catalog cards. Upload a model
photo, fill in the fabric, model/code and sizes, and download a print-ready PNG
(1472 × 2048, or ×2 high-res) that matches the brand's Illustrator template 1:1.

**Use it here:** open `index.html`, or the GitHub Pages link in the repo sidebar.

Everything runs locally in the browser — photos never leave your computer.
